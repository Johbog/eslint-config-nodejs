# eslint-config-node
Eslint config for node

## installation
`npm i -D johbog/eslint-config-nodejs`

Add the following to your package.json:

    "eslintConfig": {
      "extends": "nodejs"
    }

or create a .eslintrc.json config with:

    {
      "extends": "nodejs"
    }
